# Reliability Projects — Numerical Experiments

This folder contains Jupyter notebooks dedicated to numerical experiments in reliability theory. The main objective is to provide reproducible computational experiments supporting theoretical results.

---

## Scope of the Work

The notebooks include numerical studies related to:

- Reliability analysis of repairable systems
- Preventive maintenance policies and optimization
- Evaluation of system lifetime and failure behavior
- Threshold-based maintenance strategies
- Numerical validation of theoretical results
