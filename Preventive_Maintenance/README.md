# Numerical Experiments for Optimal Preventive Maintenance in Two-Unit Reliability Systems

This repository contains Jupyter notebooks reproducing the numerical experiments presented in the study of preventive maintenance policies for a two-unit priority standby system with repair.

The notebooks illustrate the theoretical results obtained in the associated research work and provide numerical procedures for computing:

* Threshold times beyond which maintenance becomes beneficial.
* Optimal maintenance times.

We focus on systems whose main unit exhibits either a bathtub-shaped or an upside-down bathtub-shaped hazard rate.

## Contents

* Implementation of the numerical examples presented in the paper.
* Computation of threshold and optimal maintenance times.
* Visualization of maintenance performance metrics.
* Reproducible numerical experiments.

## Associated Publication

Carballo A., Crespo Navas M., and Valdés J. E. *Preventive Maintenance of a Two-Unit Priority Standby System with Repair*, 2026.
